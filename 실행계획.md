산출물
- `실행계획.md` (이 파일)

1. 계획 문서 작성 및 사용자 확인 (현재 단계)

3. 예제 컴파일 및 실행 확인 (Windows: `g++` 사용)
4. 필요 시 추가 설명 또는 확장 예제 제공

파일 및 빌드 정보

- 컴파일 명령 예시:
```

g++ -std=c++20 friend_class.cpp -o friend_class.exe
```
